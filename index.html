💡 문제 해석 및 접근 방법

이 문제는 조합(Combination) 개념을 활용하는 문제입니다. 배열에서 서로 다른 두 인덱스의 값을 더해서 나올 수 있는 모든 경우의 수를 구해야 합니다.

핵심 포인트:

  • 중복 제거: 같은 합은 한 번만 포함
  • 이중 반복문으로 모든 조합 확인
  • 결과 배열을 오름차순 정렬

def solution(numbers): # 방법 1: 기본 접근법 (여러분이 푼 방식) result = set() # set을 사용하면 중복 자동 제거 for i in range(len(numbers)): for j in range(i + 1, len(numbers)): result.add(numbers[i] + numbers[j]) return sorted(list(result)) # 방법 2: 더 간결한 방식 (itertools 활용) from itertools import combinations def solution_advanced(numbers): return sorted(list(set(a + b for a, b in combinations(numbers, 2))))
⚡ 성능 분석 및 최적화 팁
시간 복잡도
• 이중 반복문: O(n²)
• 정렬: O(k log k)
• 전체: O(n² + k log k)
최적화 포인트
• set 사용: 중복 제거가 더 효율적
• combinations: 코드가 더 간결
• 리스트 컴프리헨션: 성능 향상
